AWS S3 Static Website Hosting

This simple website demonstrates how to host static content on AWS S3 using Terraform.

  • AWS S3 Bucket: Stores and serves static website files.
  • Versioning: Keeps track of all file changes and versions.
  • Logging: Access logs are stored in a separate S3 bucket.
  • Public Access: Configured for internet visibility with secure policies.
  • Terraform: Infrastructure as Code tool to automate AWS resource creation.

How it works:
Terraform provisions the S3 bucket, sets up website hosting, enables versioning and logging, and applies the necessary policies for public access. You can update this page by editing index.html and re-deploying with Terraform.
